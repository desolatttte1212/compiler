#  MiniCompiler

**MiniCompiler** — полнофункциональный учебный компилятор для упрощённого C-подобного языка. Проект реализует полный цикл трансляции: от лексического анализа исходного кода до генерации нативного исполняемого файла x86-64. Поддерживает сборку под Windows (`win64`) и Linux (`elf64`).

---

## ✅ Реализованный функционал

В рамках проекта последовательно реализованы все основные фазы работы компилятора:

### 1. Лексический анализ (Lexer)
- Токенизация ключевых слов, идентификаторов, числовых литералов (int/float) и строк.
- Поддержка однострочных и многострочных комментариев.
- Точное отслеживание позиций ошибок (строка и столбец).

### 2. Синтаксический анализ (Parser)
- Рекурсивный спуск для построения Абстрактного Синтаксического Дерева (AST).
- Корректная обработка приоритетов и ассоциативности арифметических и логических операторов.
- Механизм восстановления после синтаксических ошибок (Error Recovery).

### 3. Семантический анализ
- Построение таблицы символов с учётом вложенных областей видимости (Scopes).
- Проверка типов: контроль соответствия при присваивании, возврате значений и вызовах функций.
- Детектирование необъявленных переменных и дублирования объявлений.

### 4. Промежуточное представление (IR)
- Генерация SSA-подобного промежуточного кода.
- Формирование Графа потока управления (Control Flow Graph) из базовых блоков.
- Вывод IR в текстовом, JSON или DOT форматах.

### 5. Оптимизация (Optimizer)
- **Constant Folding & Propagation:** вычисление константных выражений на этапе компиляции.
- **Dead Code Elimination (DCE):** удаление неиспользуемых инструкций.
- **Peephole Optimization:** локальная оптимизация последовательностей команд.

### 6. Генерация кода (Codegen)
- Генерация ассемблерного кода x86-64 (`win64` / `elf64`).
- Управление стеком: выделение локальных переменных, выравнивание по 16 байт, Shadow Space (Windows ABI).
- Реализация Calling Convention: передача аргументов через регистры (`rcx, rdx, r8, r9` для Windows; `rdi, rsi, rdx, rcx, r8, r9` для Linux).

### 7. Тестирование и автоматизация
- Полный набор юнит- и интеграционных тестов для каждой фазы.
- Система сравнения с эталонными файлами (`.expected`).
- Утилиты для автоматического обновления эталонов при изменении формата вывода.
